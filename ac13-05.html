AC13.05 | Choisir les mécanismes de gestion de données adaptés au développement de l’outil et argumenter ses choix

Retour à la liste

Ce que j’ai fait

J'ai analysé les différentes manières de stocker et d'organiser les informations pour mes projets (fichiers JSON, bases de données, variables structurées) afin de sélectionner la solution la plus efficace pour le stockage des données.

Pourquoi je l’ai fait

Le choix du mécanisme de gestion de données impacte directement la performance et la facilité de maintenance d'une application. Il faut s'assurer que les données sont facilement accessibles, sécurisées et bien organisées.

Comment je l’ai fait

Méthode : Comparaison entre stockage persistant (fichiers) et stockage temporaire. Réflexion sur la structure des données (tableaux, dictionnaires).
Outils : Python (gestion de listes et dictionnaires), formats JSON/CSV pour le stockage.
Ressources : Cours sur les structures de données, documentations techniques.
