SAÉ 1.04 : Création d'un site web personnel

Voir AC

Ce projet consiste en la conception, l'administration et l'hébergement d'un portfolio professionnel. Il valide ma capacité à gérer un service web de A à Z.

Compétences mises en œuvre

  • Déploiement & Hébergement : Configuration d'un dépôt GitHub et déploiement via GitHub Pages.
  • Gestion de version : Utilisation des commandes git pour administrer les mises à jour.
  • Intégration Web : Adaptation d'un template Bootstrap responsive et respect W3C.

L'administration ne se limite pas aux serveurs, elle concerne aussi la gestion rigoureuse des sources et la mise à disposition de services fiables.

Apprentissages Critiques

La mise en œuvre du service d'hébergement via GitHub Pages a permis de comprendre le cycle de déploiement continu. La sécurisation a été assurée par l'activation automatique du protocole HTTPS via SSL Let's Encrypt.
