SAÉ 1.05 : Traitement de données publiques

L'objectif principal est de prouver ma capacité à automatiser la récupération et la transformation de données réseau.

  • Script d'automatisation : Développement d'un script Python capable de récupérer des données au format CSV sur data.gouv.fr.
  • Analyse & Calculs : Agrégation des données (calcul de moyennes, maximums, filtrage) et conversion au format JSON.
  • Visualisation : Transmission automatique des données à une API web (QuickChart) pour génération de graphiques.
  • Versioning : Utilisation rigoureuse de GitLab pour le stockage du code et le travail d'équipe.
