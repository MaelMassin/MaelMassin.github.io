SAÉ 1.02 : S'initier aux réseaux informatiques

Mise en œuvre complète d'une infrastructure réseau pour appréhender l'interaction entre les différents équipements.

  • Plan d'adressage : Établissement d'un plan IPv4 complet avec masques et schémas détaillés.
  • Configuration : Paramétrage de postes de travail, commutateurs et serveurs web statiques.
  • Validation : Test de connectivité et déploiement d'une page web de présentation d'entreprise.

SAÉ 1.03 : Caractérisation de systèmes de transmission

Étude technique des supports de liaison physique pour optimiser les flux de données.

  • Mesures physiques : Analyse de l'atténuation et du retard sur câbles Ethernet et Coaxiaux.
  • Simulation : Utilisation de logiciels spécialisés pour confronter les mesures aux équations théoriques.

Expérience Terrain

Matériel : Cisco (Routeurs & Switches), Câblage RJ45/Coaxial

Logiciels : Wireshark, Simulateurs de transmission
